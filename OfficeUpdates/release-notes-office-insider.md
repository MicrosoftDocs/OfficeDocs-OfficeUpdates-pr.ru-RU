# <a name="release-notes-for-office-insiders"></a>Заметки о выпуске для участников программы предварительной оценки Office

Эта статья содержит заметки о выпуске для сборок приложений Word, Excel, PowerPoint, Outlook, Access и Project для Windows, предоставляемых в рамках программы предварительной оценки. Каждую неделю мы будем сообщать об интересных новых возможностях, важных исправлениях и существенных проблемах, о которых вам следует знать. Обратите внимание на то, что развертывание возможностей (а иногда даже исправлений) для участников программы предварительной оценки зачастую занимает определенное время. Благодаря этому мы обеспечиваем стабильную работу возможностей до того, как они становятся доступны более широкой аудитории. Если вы не видите чего-либо из описанного ниже, не беспокойтесь, вы получите это позже.  

> [!NOTE]
> - Заметки о выпуске публикуются еженедельно и могут представлять собой компиляцию для нескольких сборок
> - Дата публикации заметок о выпуске может не совпадать с фактической датой выпуска сборки

Версия 1906 (сборка 11708.20006)

## <a name="whats-new"></a>Новые возможности:

### <a name="outlook"></a>Outlook

#### <a name="user-experience-updates"></a>Обновления интерфейса пользователя

Выпущены ожидавшиеся обновления, включая упрощенную ленту и визуальное изменение области папок, списка сообщений и области чтения.

##### <a name="getting-started"></a>Начало работы:

Эти изменения относятся к новому стандартному пользовательскому интерфейсу; он был доступен с помощью переключателя "Ожидается в ближайшее время" с середины декабря для 100 % рабочих сред

#### <a name="customizable-simplified-ribbon"></a>Настраиваемая упрощенная лента

Возможность простой настройки для переключения между классическим и упрощенным представлением, а также для закрепления и открепления команд.

##### <a name="getting-started"></a>Начало работы:

Пользователи могут перейти к упрощенной ленте, включив переключатель "Ожидается в ближайшее время" (изначально) и щелкнув шеврон на ленте, чтобы переключаться между классической многострочной и новой упрощенной однострочной лентой.

##### <a name="scenarios-to-try"></a>Возможные варианты:

Переключение с классической на упрощенную ленту

#### <a name="pick-your-favorite-action"></a>Выбор избранного действия

Не используете действия "Пометить" и "Удалить"? Что насчет "Архивировать" и "Пометить как прочитанные"? Настройте меню быстрых действий с помощью команд, используемых чаще всего.

##### <a name="getting-started"></a>Начало работы:

Чтобы выбрать быстрые действия, щелкните правой кнопкой мыши сообщение в списке для отображения контекстного меню. Затем выберите пункт "Задать быстрые действия"

##### <a name="scenarios-to-try"></a>Возможные варианты:

Изменение используемых по умолчанию действий "Пометить" и "Удалить" на "Архивировать", "Переместить", "Пометить как прочитанные" или "Отсутствует" для уменьшения элементов в списке сообщений

#### <a name="relaxed-or-tighter-layout-you-choose"></a>Самостоятельно выбирайте свободный или компактный макет

С помощью компактных интервалов можно выбрать дополнительное пространство между элементами или более сжатый макет, чтобы увидеть больше элементов.

##### <a name="getting-started"></a>Начало работы:

Вкладка "Вид", флажок "Уменьшить интервал" в группе "Сообщения" для классической ленты и параметры текущего представления для упрощенной ленты

##### <a name="scenarios-to-try"></a>Возможные варианты:

Использование Outlook для просмотра и создания сообщений с включенным и отключенным параметром. При включенном параметре "Уменьшить интервал" на странице располагается больше сообщений, а элементы управления в формах создания упрощаются.

#### <a name="dedupe-mru-entries-when-using-the-onedrive-sync-client"></a>Дедупликация недавно использовавшихся записей при применении клиента синхронизации OneDrive

Улучшите интеграцию с клиентом синхронизации OneDrive с помощью облачных вложений, выполнив дедупликацмию недавно использовавшихся записей, обеспечивающую ускорение вложения в виде копии для синхронизированных данных

##### <a name="getting-started"></a>Начало работы:

При использовании клиента синхронизации OneDrive дубликаты больше не отображаются в меню последних выбиравшихся для вложений файлов.

##### <a name="scenarios-to-try"></a>Возможные варианты:

Включение клиента синхронизации OneDrive и использование меню "Вложить файл" в классической версии Outlook

#### <a name="improved-shared-folder-synchronization-for-mailboxes-with-many-folders"></a>Улучшенная синхронизация общих папок для почтовых ящиков с большим числом папок

В течение нескольких лет при синхронизации общих почтовых ящиков в приложении Outlook существовало ограничение не более 500 папок. В результате этого изменения при синхронизации в Outlook больше не будет применяться это ограничение в 500 папок.

##### <a name="getting-started"></a>Начало работы:

Создайте в почтовом ящике 1000 папок, предоставьте другому пользователю доступ к почтовому ящику, создайте профиль Outlook для "другого пользователя" и убедитесь в выполнении синхронизации.

### <a name="word"></a>Word

#### <a name="erase-just-a-little-bit"></a>Стереть лишь небольшую часть

##### <a name="getting-started"></a>Начало работы:

##### <a name="scenarios-to-try"></a>Возможные варианты:
