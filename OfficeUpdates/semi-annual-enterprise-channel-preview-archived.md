# <a name="archived-release-notes-for-semi-annual-enterprise-channel-preview"></a>Архивные заметки о выпусках для Полугодового канала (предварительной корпоративной версии)

В этих заметках о выпусках содержатся сведения о новых функциях и обновлениях, не связанных с безопасностью, включенных в обновления Полугодового канала (предварительная корпоративная версия) для Office 365 профессиональный плюс, Visio Pro для Office 365, классического клиента Project Online и Office 365 бизнес.

> [!NOTE]
> - Мы часто развертываем функции (а иногда даже исправления) для Полугодового канала (предварительная корпоративная версия) по истечении определенного времени. Если у вас пока нет каких-либо из перечисленных ниже возможностей, они скоро появятся. [Подробнее](https://support.office.com/article/when-do-i-get-the-newest-features-in-for-office-365-da36192c-58b9-4bc9-8d51-bb6eed468516?ui=en-US&rs=en-US&ad=US)
> - Microsoft Teams входит в состав новых установок Полугодового канала (предварительная корпоративная версия) с версии 1902. Приложение Teams добавляется в существующие установки Полугодового канала (предварительная корпоративная версия) при их обновлении до версии 1908 или более поздней. Дополнительные сведения см. в статье [Развертывание Microsoft Teams с Office 365 профессиональный плюс](/deployoffice/teams-install).

### <a name="resolved-issues"></a>Решенные проблемы
### <a name="access"></a>Access

- Исправлена проблема, из-за которой запрос на объединение, включающий ссылки на удаленные таблицы (например, таблицы SQL Server), может приводить к закрытию и перезапуску Access.

- Исправлена проблема, из-за которой агрегатная функция, например, Sum, может обрезать результат до целого значения.

### <a name="excel"></a>Excel

- Исправлена проблема, из-за которой выбор ячейки после прокрутки мог приводить к выбору неправильной ячейки.

- Исправлена проблема, при которой для фильтра сводной таблицы OLAP было задано значение, удаленное из куба.

- Это изменение обходит проблему, связанную с некоторыми графическими драйверами Intel, благодаря использованию программной отрисовки.

- Исправлена проблема, из-за которой функция Lookup может возвращать ошибку.

- Значительно повышена производительность при удалении столбцов с объединенными ячейками.

- Исправлена проблема, которая была причиной ошибки времени выполнения макроса при активации свернутых окон.

### <a name="outlook"></a>Outlook

- Исправлена проблема с выбором алгоритма SMIME.

- Исправлена проблема, из-за которой при открытии диалогового окна "Правила" появлялось сообщение &quot;Правила на этом компьютере противоречат правилам Microsoft Exchange&quot;.

- Исправлена проблема, которая была причиной доступа веб-надстроек к сообщениям с управляемыми цифровыми правами, когда это не предполагалось.

### <a name="word"></a>Word

- Исправлена проблема с отслеживанием изменений, которые иногда переходят в режим бесконечного цикла. 

### <a name="office-suite"></a>Набор Office

- Исправлена проблема, из-за которой не удавалось правильно повернуть полуширинные символы катаканы при использовании вертикальных текстовых полей в PowerPoint.

- Решена проблема, из-за которой обновления Office могли неожиданно скачать файлы из Office CDN вместо намеченного источника, такого как локальная или сетевая папка или расположение, предоставленное Configuration Manager.

- Чтобы защитить пользователей Office, обновления Microsoft Office теперь подписываются исключительно с использованием алгоритма SHA-2.

### <a name="feature-updates"></a>Обновления функций
### <a name="access"></a>Access

- **Изменение масштаба стало удобнее.** Увеличивайте поле масштаба и меняйте шрифт — все эти параметры сохранятся. [Подробнее](https://support.office.com/article/9a62d165-67f8-4790-bad8-a2c2e83dcedf)

- **Отслеживание объектов базы данных.** Не теряйте из виду активную вкладку, легко перетаскивайте вкладки для изменения их порядка и закрывайте объекты базы данных одним щелчком мыши.

- **Удобное переключение.** Новый диспетчер учетных записей отображает все ваши рабочие и личные учетные записи Office 365 в одном месте. Переключаться между ними стало еще проще. [Подробнее](https://support.office.com/article/d17de17e-9f26-4dae-bd08-d7b8f46eb0b9)

### <a name="excel"></a>Excel

- **Узнайте больше о своих данных.** Новая кнопка "Идеи" позволяет находить закономерности в данных и предлагает интеллектуальные, индивидуальные решения на их основе. [Подробнее](https://support.office.com/article/3223aab8-f543-4fda-85ed-76bb0295ffc4)

- **Быстрый поиск файла.** Ищете файл, с которым недавно работали? Просто воспользуйтесь полем поиска на странице "Файл" > "Главная", чтобы найти нужный файл.

- **Расширение доступа к содержимому.** Нужно сделать презентации доступными для людей с ограниченными возможностями? Доверьте это средству проверки читаемости, которое не будет вас отвлекать. Проверьте, как это работает, выбрав пункт меню Рецензирование > Проверка читаемости, и получайте требующие вашего внимания уведомления в строке состояния.

- **Удобное переключение.** Новый диспетчер учетных записей отображает все ваши рабочие и личные учетные записи Office 365 в одном месте. Переключаться между ними стало еще проще. [Подробнее](https://support.office.com/article/d17de17e-9f26-4dae-bd08-d7b8f46eb0b9)

- **Добавление живых красок в таблицы.** Вставляйте в книгу анимированные трехмерные изображения, чтобы показать, как бьется сердце, вращается планета или рычит тираннозавр. [Подробнее](https://support.office.com/article/6f08009a-3da5-400d-a706-8e23f304cd72)
