# <a name="release-notes-for-office-2016-for-mac"></a>Заметки о выпуске Office 2016 для Mac

Office 2016  для Mac — это версия Office, которая доступна для единовременной покупки в розничном магазине или по корпоративной лицензии. Она будет поддерживаться обновлениями безопасности и исправлениями ошибок, при необходимости, до 13 октября 2020 г. Минимальная поддерживаемая Office 2016 для Mac версия  macOS — 10.10 (Yosemite).

> [!NOTE]
> - В целях обеспечения безопасности Майкрософт не рекомендует использование SHA-1. [Подробнее](https://docs.microsoft.com/en-us/security-updates/SecurityAdvisories/2017/4010323)
> - Заметки о выпусках Office 2016 для Mac до сентября 2018 года см. в статье [Заметки о выпусках Office для Mac](release-notes-office-for-mac.md).  
> - Для получения дополнительной информации об Office для Mac, о версии, которая предоставляется с подписками на Office 365, см. разделы [заметки о выпуске](release-notes-office-for-mac.md) и [история обновлений](update-history-office-for-mac.md) для Office для Mac.
> - Office 2019 для Mac является наиболее свежей версией, доступной для единовременной покупки. Для получения информации о выпуске Office 2019 для Mac, см. разделы [заметки о выпуске](release-notes-office-for-mac.md) и [история обновлений](update-history-office-for-mac.md) для Office для Mac.

Обновления для Office 2016 для Mac доступны в автоматическом обновлении (Майкрософт) (MAU). Чтобы использовать автоматическое обновление (Майкрософт), запустите приложение Office, например Word, и выберите **Справка** > **Проверить наличие обновлений**. Если вы являетесь администратором и вам необходимо обновить компьютеры в вашей организации при помощи существующих инструментов развертывания программного обеспечения, можно загрузить и развернуть пакеты обновлений для каждого приложения.
