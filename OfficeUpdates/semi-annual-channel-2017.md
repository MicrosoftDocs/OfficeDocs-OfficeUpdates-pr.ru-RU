> - Следующие также сведения о новых функциях, обновления безопасности и других обновлений для Visio Pro для Office 365 и Project Online настольный клиент.
> - Эти сведения также относится к Office 365 для бизнеса, являющийся версии Office, которое поставляется с некоторыми планов Office 365, например бизнеса расширенный.

[Дополнительные сведения](https://blogs.office.com/2017/03/06/new-in-access-2016-large-number-bigint-support/)

### <a name="excel-feature-updates"></a>Excel: Обновляет функции
-   **Поддержка защиты информации Windows (НЗП):**   Excel теперь enlightened приложения и теперь можно различить организации и конфиденциальных данных, правильно, определение которого необходимо защитить, на основе настроенного политик.  [Дополнительные сведения](https://aka.ms/wiptechnet)
-   **Получить & преобразования улучшения:** В редакторе запросов создайте новый столбец, предоставляя примеры значений. При вводе Excel определяет необходимые преобразования и показывает предварительный просмотр нового столбца.
-   **Вставка последние ссылки:** Легко вложения гиперссылок для новых файлов на основе облака или веб-сайтов и создание удобной для восприятия отображаемые имена людей, с помощью программы чтения с экрана. [Дополнительные сведения](https://support.office.com/article/002684c4-bf06-422b-821f-b4ef84fdb0e3)
-   **Настраивать макет сводной таблицы по умолчанию:** Настройка сводной таблицы способ like и начать с макета каждый раз при создании новой сводной таблицы. [Дополнительные сведения](https://support.office.com/article/efd8569c-f07a-43c1-9db2-4f2912a0f94e)
-   **Получить & преобразования улучшения:** Пользователи могут создавать новые столбцы, пример и разбить на строки столбцов таблицы. Указание параметров для SAP HANA и группировки данных теперь проще.
-   **Централизованный развертывания надстроек**: Администраторы могут развертывание и обновление надстроек для пользователей или групп из центра администрирования Office 365. [Дополнительные сведения](https://dev.office.com/docs/add-ins/publish/centralized-deployment)
-   **Настройки панели быстрого доступа:** Значки индекс и надстрочный знак можно добавить на панель быстрого доступа.
-   **Получить & преобразования улучшения:** Автоматическое обнаружение символ разделителя, если разбиение столбцов с помощью редактора запросов, выберите какие пример файла будет использоваться с объединить двоичные файлы и укажите коллекции пакет для подключения к при использовании соединителя DB2.
-   **Шрифта Дубаи:** Семейство шрифтов, поддерживающего западных языков как, так и основных языков, использующих арабского языка сценариев. [Дополнительные сведения](https://support.office.com/article/c862df16-ae0d-46d9-b117-aa3f41f9706e)
-   **Фона удаления:** Удаление фона рисунков с помощью средства рисования свободной форме.
-   **Получить & преобразования улучшения:** Использование «Выбор связанных таблиц» в диалоговом окне Навигатор с ODCB и OLEDB соединители, объединения нескольких файлов непосредственно из диалогового окна предварительного просмотра данных папки и использование нового пункта контекстного меню в окно редактора запросов для вставки нового действия в существующих запросов.
-   **Использовать перо для выбора и изменения объектов:** Скопировать маркеры объектов с цифровыми перо, чтобы изменить размер, поворот, перемещение и многое другое.
-   **Сопоставление диаграммы:** Сравнение значений и Показать категории по географическим регионам. [Дополнительные сведения](https://support.office.com/article/f2cfed55-d622-42cd-8ec9-ec8a358b593b)
-   **Изображения SVG:** Вставка и редактирование масштабируемой векторной графики (SVG) в книгах. [Дополнительные сведения](https://support.office.com/article/69f29d39-194a-4072-8c35-dbe5e7ea528c)
-   **Вставлять значки:**  Используйте значки из библиотеки standard масштабируемой векторной графики (SVG) файлов, перейдя для вставки \> иллюстрации \> значки. [Дополнительные сведения](https://support.office.com/article/e2459f17-3996-4795-996e-b9a13486fa79)
-   **Сохранить последние папок:** Сохранение книги на недавно использовавшихся папок с помощью последние вкладки при переходе в файл \> Сохранить как.
-   **Улучшения читаемости:** Улучшенная поддержка с помощью клавиатуры, "Экранный диктор" и других вспомогательных чтение и редактирование книг. [Дополнительные сведения](https://support.office.com/article/51fcb17a-b15b-4b13-ae04-d4f38ece3f78)

### <a name="excel-non-security-updates"></a>Excel: Обновления не безопасности
-   Устранение проблемы с копирования и вставки в Excel, таких как без учета табуляции HTML-текста, неправильно Удаление пустых строк и столбцов, без учета выравнивание и формул без учета.
-   Устранение неполадок расположения ошибки при попытке сохранить изменения в документы, синхронизируются с клиентом OneDrive.
-   Устранить проблему, где Excel завершается со сбоем, при добавлении данных таблицы в модели данных и темы высокой контрастности устанавливается черный.
-   Устранение неполадок где Excel выходит из строя для книги, содержащие внешние ссылки.
-   Устранение неполадок где Вставка ячеек из Excel в Microsoft Word отображает нули в ячейках, несмотря на то, что не выбран параметр «Показать нуль в ячейках, которые содержат нулевые значения».
-   Устранение неполадок где Excel не устанавливает пароль защиты листа при применении программным способом для книг, созданных в Excel 2010 и более ранних версий.
-   Устранение неполадок где центр и объединения не работает в сгруппированных листа.
-   Устранение неполадок которых отсутствуют новые функции, которые были представлены после выпуска Office 2016 — например, TEXTJOIN, ОБЪЕДИНИТЬ, IFS, MAXIFS и MINIFS.
-   Устранение неполадок где Excel прекращает работу при попытке пользователя для применения разрешений на уровне ячеек.
-   Устранение неполадок где сохранение больших файлов в OneDrive для бизнеса вызывает файл к блокированию и пользователь не может изменить файл, пока пользователь закроет и повторно открывает Excel.
-   Устранение неполадок место новое окно, где отображается серым при открытии книги .xls.
-   Устранение неполадок где Excel может привести к сбою при вставке гиперссылки.
-   Устранение неполадок где Excel не удается при добавлении карты XML.
-   Устранение неполадок где кнопки команды для надстройка не работает после обновления надстройки или после удаления и попытку загрузки надстройки из магазина Office.
-   Устранение неполадок где Excel может привести к сбою при работе с DLL-Библиотека стилей с помощью VBA.
-   Устранение неполадок где Excel прекращает работу при выборе в форму элемент управления раскрывающийся список на листе диаграммы.
