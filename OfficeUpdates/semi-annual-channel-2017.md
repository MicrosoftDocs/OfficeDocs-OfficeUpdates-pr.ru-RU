> - Следующие также сведения о новых функциях, обновления безопасности и других обновлений для Visio Pro для Office 365 и Project Online настольный клиент.
> - Эти сведения также относится к Office 365 для бизнеса, являющийся версии Office, которое поставляется с некоторыми планов Office 365, например бизнеса расширенный.

*Версия 1705 (сборка 8201.2193)*

### <a name="access-feature-updates"></a>Access. Обновления функций
-   **Диалоговое окно "Что нового".** Когда приложение Access открывается после обновления, появляется диалоговое окно с описанием новых функций. Кроме того, это окно можно открыть, последовательно выбрав "Файл" \> "Учетная запись" \> "Что нового".
-   **Поддержка типа данных bigint (больших чисел).** Используйте тип данных "Большое число" в таблицах Access для оперирования большими числами при вычислениях, для связи с внешней базой данных, в которой применяется эквивалентный тип данных (например, bigint в SQL Server), а также для импорта содержимого такой базы данных.[Дополнительные сведения](https://blogs.office.com/2017/03/06/new-in-access-2016-large-number-bigint-support/)

### <a name="excel-feature-updates"></a>Excel. Обновления функций
-   **Поддержка Windows Information Protection (WIP).** Приложение Excel теперь различает корпоративные и персональные данные, правильно определяя те из них, которые подлежат защите, на основе настроенных политик.[Дополнительные сведения](https://aka.ms/wiptechnet)
-   **Улучшение раздела "Скачать & преобразовать".** Создайте новый столбец в редакторе запросов, предоставив примеры значений. По мере ввода текста приложение Excel определяет, какие преобразования необходимы, и отображает предварительную версию нового столбца.
-   **Вставка ссылок на недавние элементы.** Легко добавляйте гиперссылки на недавние облачные файлы или веб-сайты и создавайте понятные отображаемые имена для людей, использующих средства чтения с экрана.[Дополнительные сведения](https://support.office.com/article/002684c4-bf06-422b-821f-b4ef84fdb0e3)
-   **Персонализация макета сводной таблицы по умолчанию.** Настройте сводную таблицу по своему вкусу. Начните с макета, который вы каждый раз создаете. [Дополнительные сведения](https://support.office.com/article/efd8569c-f07a-43c1-9db2-4f2912a0f94e)
-   **Улучшения раздела "Скачать & преобразовать":** Пользователи могут создавать новые столбцы по образцу и разбивать столбцы таблицы на строки. Стало проще указывать параметры для SAP HANA и группировать данные.
-   **Централизованное развертывание надстроек**. Администраторы могут развертывать и обновлять надстройки для пользователей или групп в Центре администрирования Office 365. [Дополнительные сведения](https://dev.office.com/docs/add-ins/publish/centralized-deployment)
-   **Настройка панели быстрого доступа.** Возможность добавления на панель подстрочных и надстрочных значков.
-   **Улучшения раздела "Скачать & преобразовать":** Доступно автоматическое обнаружение символа разделителя при разделении столбцов с помощью редактора запросов. Можете выбрать, какой пример файла использовать с параметром "Объединить двоичные данные", а также указать коллекцию пакетов для подключения при использовании соединителя DB2.
-   **Шрифт Dubai.** Семейство шрифтов, поддерживающее как западноевропейские языки, так и основные языки, использующие арабскую вязь. [Дополнительные сведения](https://support.office.com/article/c862df16-ae0d-46d9-b117-aa3f41f9706e)
-   **Удаление фона.** Удаляйте фон изображения с помощью инструмента рисования в свободной форме.
-   **Улучшения раздела "Скачать & преобразовать":** Указывайте элемент "Выбор связанных таблиц" в диалоговом окне "Навигатор" с соединителями ODCB и OLEDB, объединяйте несколько файлов прямо в диалоговом окне "Просмотр данных" для папки, используйте новый параметр контекстного меню в окне "Редактор запросов", чтобы вставить новые шаги в существующие запросы.
-   **Выбор и изменение объектов с помощью пера.** Захватывайте маркеры с помощью цифрового пера, чтобы вращать и перемещать объекты, менять их размеры и выполнять другие действия.
-   **Диаграмма с картой.** Сравнивайте значения и показывайте категории по географическим регионам.[Дополнительные сведения](https://support.office.com/article/f2cfed55-d622-42cd-8ec9-ec8a358b593b)
-   **Изображения SVG.** Вставляйте и редактируйте изображения SVG в книгах. [Дополнительные сведения](https://support.office.com/article/69f29d39-194a-4072-8c35-dbe5e7ea528c)
-   **Вставка значков.** Чтобы использовать значки из стандартной библиотеки файлов SVG, выберите "Вставка" \> "Иллюстрации" \> "Значки". [Дополнительные сведения](https://support.office.com/article/e2459f17-3996-4795-996e-b9a13486fa79)
-   **Сохранение в недавних папках.** Чтобы сохранить книгу в недавней папке, используйте вкладку "Последние" в меню "Файл" \> "Сохранить как".
-   **Улучшены специальные возможности.** Улучшена поддержка чтения и редактирования книг с помощью клавиатуры, экранного диктора и других специальных возможностей. [Дополнительные сведения](https://support.office.com/article/51fcb17a-b15b-4b13-ae04-d4f38ece3f78)

### <a name="excel-non-security-updates"></a>Excel. Обновления, не связанные с безопасностью
-   Исправлены ошибки, возникающие при копировании данных в Excel, например игнорирование текста в формате HTML с разделителями табуляции, неправильное удаление пустых строк и столбцов, игнорирование выравнивания и формул.
-   Устранена проблема, приводящая к возникновению ошибок при попытке сохранить изменения в документах, которые синхронизируются с клиентом OneDrive.
-   Исправлена ошибка, из-за которой Excel дает сбой при добавлении данных листа в модель данных, когда выбрана черная высококонтрастная тема.
-   Исправлена ошибка, из-за которой Excel дает сбой при работе с книгами, содержащими внешние ссылки.
-   Исправлена ошибка, из-за которой в ячейках, вставленных из Excel в Word, отображаются нули, даже если параметр "Показывать нули в ячейках, которые содержат нулевые значения" не выбран.
-   Решена проблема, из-за которой в Excel не удается программно задать защиту листа паролем для книг, созданных в Excel 2010 или более ранней версии.
-   Исправлена проблема, из-за которой элемент "Объединить и выровнять по центру" не удается использовать на сгруппированных листах.
-   Устранена проблема, из-за которой отсутствуют новые функции, представленные после выпуска Office 2016 (например, ОБЪЕДИНИТЬ, СЦЕП, ЕСЛИМН, МАКСЕСЛИМН и МИНЕСЛИМН).
-   Устранена проблема, из-за которой Excel дает сбой, когда пользователь пытается применить разрешения уровня ячейки.
-   Устранена проблема, из-за которой при сохранении большого файла в OneDrive для бизнеса происходит блокировка этого файла, а пользователь не может отредактировать его до тех пор, пока не перезапустит Excel.
-   Устранена проблема, из-за которой новое открывающееся окно затемненное, если открыта книга XLS.
-   Устранена проблема, из-за которой добавление гиперссылки может привести к сбою Excel.
-   Исправлена ошибка, в результате которой добавление карт XML может привести к сбою Excel.
-   Устранена проблема, из-за которой кнопка для надстройки не работает после обновления этой надстройки или после ее удаления и повторного скачивания в Магазине Office.
-   Исправлена ошибка, в результате которой работа с таблицами DLL с помощью VBA может привести к сбою Excel.
-   Устранена проблема, из-за которой Excel дает сбой, если выбрать элемент управления формы "поле со списком" на листе диаграммы.
