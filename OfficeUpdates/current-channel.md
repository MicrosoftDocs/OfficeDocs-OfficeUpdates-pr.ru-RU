# <a name="release-notes-for-current-channel-releases-in-2020"></a>Заметки о выпуске для текущих выпусков канала в 2020

Эти заметки о выпусках содержат сведения о новых функциях и обновлениях, не связанных с безопасностью, которые входят в текущие обновления канала 2020 для приложений Microsoft 365 для предприятий, Microsoft 365 для бизнеса, а также версии подписки для приложений Project и Visio для настольных ПК.

> [!IMPORTANT]

Дополнительные сведения см. в [этой статье](https://go.microsoft.com/fwlink/p/?linkid=2127441).
