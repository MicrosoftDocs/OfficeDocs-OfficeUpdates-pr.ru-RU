# <a name="release-notes-for-semi-annual-enterprise-channel-preview-releases-in-2020"></a>Заметки о выпусках Полугодового канала (предварительной корпоративной версии) в 2020 г.

Эти заметки о выпусках содержат информацию о новых возможностях и обновлениях, не связанных с безопасностью, которые включены в Полугодовой канал (предварительная корпоративная версия) в 2020 г. для подписок "Приложения Microsoft 365 для предприятий", "Приложения Microsoft 365 для бизнеса", а также эквивалентов классических приложений для Project и Visio, предоставляемых по подписке.

> [!IMPORTANT]
> Мы вносим некоторые изменения в каналы обновления для Приложений Microsoft 365, в том числе добавляем новый канал обновления (Monthly Enterprise Channel) и переименовываем существующие каналы обновления. Дополнительные сведения см. в [этой статье](https://go.microsoft.com/fwlink/p/?linkid=2127441).

### <a name="resolved-issues"></a>Решенные проблемы
### <a name="excel"></a>Excel

- Ускорение загрузки файлов, доступных в локальной папке OneDrive.

- Исправлена проблема, из-за которой CustomUI XML для настраиваемой вкладки ленты удалялся при сохранении в SharePoint или OneDrive.

- Исправлена проблема, из-за которой появлялось сообщение об ошибке "На эту книгу имеются ссылки из других книг, поэтому закрыть ее нельзя", так как надстройки загружались в алфавитном, а не в указанном пользователем порядке.

- Исправлена проблема, при которой книга могла быть скрыта при щелчке гиперссылки на определенное место в уже открытой книге.

- Исправлена проблема, из-за которой некоторые скопированные и вставленные ссылки диаграмм использовали сопоставленные адреса дисков вместо универсальных адресов.

- Повышена производительность при удалении столбцов с объединенными ячейками.

- Устранена проблема, из-за которой имена принтеров могли повторяться в списке принтеров в диалоговом окне "Печать".

- Исправлена проблема с задержками при сохранении файлов с листами, содержащими несколько формул с определенными именами.

- Устранена проблема, при которой повторяющиеся встречи или собрания отображались с ошибкой во времени при приближении изменений в определении часового пояса.

- Устранена проблема, из-за которой категории иногда исчезали из сообщений электронной почты.

- Устранена проблема, из-за которой представители видели разные иерархии папок общих почтовых ящиков на разных компьютерах.

- Исправлена проблема, из-за которой у представителей при редактировании существующей встречи в календаре руководителя отображалось сообщение об ошибке.

- Исправлена проблема, из-за которой кодировка текста отчета о недоставке изменялась с Юникода на ASCII после изменения темы.

- Устранена проблема, вызывавшая сбой после того, как две надстройки добавляли кнопку в одну и ту же группу ленты.

- Исправлена проблема, из-за которой пользователи не могли отправлять электронные сообщения по адресам из личного списка рассылки.

- Устранена проблема, из-за которой нельзя было добавить ссылки в сообщения электронной почты с правильным клиентским разрешением по умолчанию, если для этого разрешения был установлен параметр "все".

- Устранена проблема, связанная с тем, что пользователи не могли сохранять вложения OneDrive вне клиента на локальный компьютер при сохранении через диалоговое окно "Безопасность".

### <a name="word"></a>Word

- Исправлена проблема с совместной работой над документом при включении политики FileBlick\Word2007Files.

- Устранена проблема, при которой функция экспресс-блоков Word не работала при добавлении переименованного поля подстановки.

### <a name="office-suite"></a>Набор Office

- Исправлена проблема, из-за которой при совместной работе над объемными файлами PowerPoint чрезмерно повышалась нагрузка на сеть и ЦП.

- Мы бэкпортировали новую версию AppV51, чтобы исправить регрессию в предыдущей версии AppV51.

- Устранена проблема со сбоем в работе узла Office в Windows при активации надстройки, когда значение реестра TabProcGrowth относилось к типу REG_SZ.

- Улучшена синхронизация и стабильность службы путем временного отключения записи видео в приложении в OneNote 2016. Это мера не влияет на локальные записные книжки.

- Улучшена синхронизация и стабильность службы путем временного уменьшения максимально допустимого размера новых внедренных вложений до 50 МБ в OneNote 2016. В случае файлов большего размера у пользователей будет возможность отправить файл в OneDrive и вставить ссылку в OneNote.

- Это обновление устраняет проблему в Microsoft Word, из-за которой текст длиной более 255 символов, вставленный при применении метки конфиденциальности, впоследствии не удавалось идентифицировать и удалить путем изменения или удаления метки, если в рамках политики меток был применен колонтитул или подложка.

- Это исправление обновляет конечную точку URL-адреса конфигурации усиленной безопасности (ECS). В результате вызова этой новой конечной точки повышается успешность получения данных из ECS.

### <a name="excel"></a>Excel

- В ряде случаев приложение Excel аварийно завершало работу при повторном открытии книги, внедренной в Word или PowerPoint.

- В некоторых случаях при сохранении в виде CSV-файла Excel мог объединять все столбцы в один столбец.

- Использование Range.ClearContents для диапазона в защищенном листе могло занимать больше времени, чем ожидалось.

- Устранена проблема с производительностью при создании диаграмм на основе шаблонов.
