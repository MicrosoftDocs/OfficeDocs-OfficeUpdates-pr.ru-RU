# <a name="release-notes-for-semi-annual-enterprise-channel-releases-in-2020"></a>Заметки о выпуске для полугодовых выпусков корпоративных каналов в 2020

Эти заметки о выпусках содержат сведения о новых функциях и обновлениях, не связанных с безопасностью, которые включены в полугодовые обновления корпоративного канала в 2020 для приложений Microsoft 365 для предприятий, Microsoft 365 для бизнеса и версии подписки для приложений Project и Visio.

> [!IMPORTANT]
> Мы вносим некоторые изменения в каналы обновления для Приложений Microsoft 365, в том числе добавляем новый канал обновления (Monthly Enterprise Channel) и переименовываем существующие каналы обновления. Дополнительные сведения см. в [этой статье](https://go.microsoft.com/fwlink/p/?linkid=2127441).

> [!NOTE]
>
>- Теперь OneNote 2016 будет включен по умолчанию, когда пользователь на полугодии корпоративного канала скачивает и устанавливает приложения Microsoft 365 в Windows 10 с портала Office.
