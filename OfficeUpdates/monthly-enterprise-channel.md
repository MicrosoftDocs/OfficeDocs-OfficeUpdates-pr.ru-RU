# <a name="release-notes-for-monthly-enterprise-channel"></a>Заметки о выпусках Ежемесячного канала (корпоративного)

Эти заметки о выпусках содержат информацию о новых возможностях и обновлениях, не связанных с безопасностью, которые включены в обновления Ежемесячного канала (корпоративного) для Приложений Microsoft 365 для предприятий, Приложений Microsoft 365 для бизнеса, а также классических приложений Project и Visio, предоставляемых по подписке.

### <a name="outlook"></a>Outlook

- Изменения происходят автоматически в шлюзе изменений, который можно быстро отключить при возникновении проблем.


- Мы добавили раздел реестра, отключающий новые возможности поиска помещений (такой же, как в Outlook в Интернете) и включающий устаревшую версию поиска помещений с предлагаемыми интервалами времени.

   Раздел реестра:

    >HKCU\SOFTWARE\Microsoft\Office\16.0\Outlook\Options\Calendar REG_DWORD “ShowLegacyRoomFinder”

    >0 (по умолчанию) — Outlook использует новый интерфейс поиска помещений на платформе OWA, когда пользователь нажимает кнопку "Поиск помещений" для поиска доступных помещений</br>
    >1 — Outlook использует устаревший пользовательский интерфейс поиска помещений для поиска доступных помещений


- Это изменение позволяет пользователям отправлять отзывы через новую систему обратной связи.


- Исправлена проблема, из-за которой параметр обратной связи был отключен для пользователей предварительной версии Office Perpetual 2021.


- Исправлена проблема, из-за которой при выборе параметра "Открыть свойства Outlook" в контекстном меню для получателя письма появлялось сообщение об ошибке.


- Исправлена проблема, из-за которой у некоторых пользователей приложение неожиданно закрывалось при загрузке карточек пользователей.


- Исправлена проблема, из-за которой пользователи сталкивались с непредвиденным закрытием при удалении папок из архивного хранилища.


- Исправлена проблема, из-за которой некоторые инструкции для функции "Сократить длительность собраний" были отключены при использовании технологий чтения с экрана.


- Исправлена проблема, из-за которой пользователи получали неожиданный запрос на изменение свойств при закрытии сообщения, на которое они ответили или которое они переадресовали.


- Исправлена проблема, которая может вызывать неожиданное закрытие при взаимодействии с Почтой Outlook или представлениями календаря.


### <a name="powerpoint"></a>PowerPoint

### <a name="word"></a>Word

- Исправлена проблема, из-за которой не открывается область редактора.


- Исправлена проблема, из-за которой в контекстных карточках холста для орфографии и грамматики отображались кнопки значков без подсказок.


### <a name="office-suite"></a>Набор приложений Office

- Исправлена проблема локализации. Теперь варианты en-gb, fr-ca и es-mx будут совпадать с соответствующими родительскими версиями.
