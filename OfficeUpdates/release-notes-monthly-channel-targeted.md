# <a name="release-notes-for-office-monthly-channel-targeted"></a>Заметки о выпуске для канала Office Monthly Channel (Targeted)

Эта статья содержит заметки о выпуске для сборок Monthly Channel (Targeted) приложений Word, Excel, PowerPoint, Outlook, Access и Project для Windows. Каждую неделю мы будем сообщать об интересных новых возможностях, важных исправлениях и существенных проблемах, о которых вам следует знать. Обратите внимание, что мы часто выпускаем функции (а иногда даже исправления) для Monthly Channel (Targeted) по истечении определенного времени. Благодаря этому мы обеспечиваем стабильную работу возможностей до того, как они становятся доступны более широкой аудитории. Если вы не видите чего-либо из описанного ниже, не беспокойтесь, вы получите это позже.  

> [!NOTE]
> - Дата публикации заметок о выпуске может не совпадать с фактической датой выпуска сборки.
> - Microsoft Teams для существующих установок Office 365 профессиональный плюс — с конца июня Microsoft Teams будет добавляться в существующие установки Office 365 профессиональный плюс (и Office 365 бизнес) при обновлении этих установок. Дата добавления приложения Teams зависит от используемого канала обновления. Дополнительные сведения см. в статье [Развертывание Microsoft Teams с Office 365 профессиональный плюс](https://docs.microsoft.com/deployoffice/teams-install).

### <a name="resolved-issues"></a>Решенные проблемы
### <a name="excel"></a>Excel

- Пользователи могут столкнуться с ошибкой при сохранении изменений, если используются некоторые наборы символов не из английского языка.

- Редактирование формул динамического массива в ячейке может привести к выравниванию текста за границами ячейки.

- Функция "Текст в столбец" может не работать для некоторых вариантов локализации.

- Исправлена проблема, из-за которой настройка ленты не загружалась при открытии внедренной книги.

- Пользователи могут столкнуться с ошибкой при доступе к скрытому именованному диапазону.

- Раскрывающееся меню полей может отображаться неправильно.

- Отключение аппаратного ускорения графики при использовании разрешения 4K может приводить к задержке отображения ячеек при прокрутке.

- Это изменение обходит проблему, связанную с некоторыми графическими драйверами Intel, благодаря использованию программной отрисовки.

### <a name="onenote"></a>OneNote

- Приложение OneNote может не открываться при использовании надстройки "Заметки к собранию" в Outlook.

### <a name="outlook"></a>Outlook

- Периодические сбои при работе с содержимым разных папок.

- Иногда изменяются размеры изображений, вставленных в сообщения электронной почты Outlook.

- Добавлена возможность применять стандартную настройку подписи S/MIME с помощью групповой политики.

- Встроенные изображения могут отображаться в размере меньше предполагаемого.

- Метки политики хранения могут отображать срок хранения в круглых скобках.

- Исправлена проблема, из-за которой подсказки политики не отображались при использовании другого отправителя.

- В карточках контактов может появляться пустое место при использовании японского языкового пакета.

- Исправлена проблема, из-за которой место проведения собрания неожиданно вновь добавлялось к собранию после его очистки.

### <a name="powerpoint"></a>PowerPoint

- Курсор может исчезнуть после перемещения фокуса с текста.

- Безопасные ссылки, ведущие из одного приложения Office в другое, могут не запускать связанное приложение.

- В некоторых случаях прокрутка на сенсорных устройствах не работает.

- Если на слайде пользователя в облачном файле имеется два (или более) видеоролика, видеоизображения отображаются правильно, но когда пользователь щелкает каждое из них для воспроизведения, видеоконтент оказывается одинаковым.

- Раскрывающееся меню полей может отображаться неправильно.

### <a name="project"></a>Project

- Если используется темный режим при переходе в панель инспектора в задаче, содержащей ресурс с превышением доступности, вы не сможете прочесть таблицу.

- У пользователей может возникать ошибка лицензирования.

- Возможна установка неверной даты кнопкой "Сегодня" в средстве выбора даты.

- Приложение Project может аварийно завершать работу при использовании функции сравнения проектов.

- Настройка трудозатрат для задач без заданий округляется до одного дня.

### <a name="word"></a>Word

- При выборе маркера примечаний теперь должна отображаться современная область примечаний, если она скрыта в переключателе области.

- Иногда щелчок правой кнопкой мыши не выделяет все слово.

- Безопасные ссылки, ведущие из одного приложения Office в другое, могут не запускать связанное приложение.

- Диспетчер стандартных блоков может отображать неправильное оповещение: &quot;Были изменены стили, стандартные блоки&quot;.

- В некоторых темах иногда сложно определить выбранное примечание.

- Курсор может оставаться активным в объекте после преобразования в предлагаемый формат.

- В некоторых ситуациях изображения в сообщениях могут быть неправильно масштабированы.

- Область примечаний иногда перезагружается при использовании команд копирования и вставки.

- Примечания иногда вставляются в неправильном порядке.

- При упоминании пользователя в карточке примечания может отображаться формат JSON.

- Раскрывающееся меню полей может отображаться неправильно.

- Изменение границы разделенного экрана может приводить к дополнительному разделению экрана.

- При применении к существующим документам шаблона, состоящего из многоуровневого списка с пользовательскими стилями, стили могут не сохраняться при определенных условиях.

- Может не работать сохранение файла после слияния при определенных условиях.

### <a name="office-suite"></a>Набор Office

- Вставка диаграммы из Excel в PowerPoint может уменьшать размер диаграммы.

- При ответе на примечание возможно вертикальное расширение текстового поля за край области.

- В продуктах на японском языке имя и фамилия пользователя учетной записи могут отображаться в неправильном порядке.

- Исправление ошибки с параметром крайнего срока обновления в GPO и средстве ODT, из-за которой относительный крайний срок соблюдается только при его первой настройке. Это исправление позволяет применять относительный крайний срок для последующих обновлений.

- При наведении указателя мыши на примечание может появляться рамка вокруг примечания.

- Решена проблема, из-за которой обновления Office могли неожиданно скачать файлы из Office CDN вместо намеченного источника, такого как локальная или сетевая папка или расположение, предоставленное Configuration Manager.

- Исправлена проблема, из-за которой удаление листов, содержащих спарклайны со ссылками на данные других листов, могло привести к обозначению файла как поврежденного при повторном открытии.

- Исправлена проблема, из-за которой выбор ячейки после прокрутки мог приводить к выбору неправильной ячейки.

- Обнаружена проблема, которая может влиять на синхронизацию локального ресурса с облачным.

### <a name="outlook"></a>Outlook
