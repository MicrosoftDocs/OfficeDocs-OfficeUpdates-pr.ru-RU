# <a name="release-notes-for-office-monthly-channel-targeted"></a>Заметки о выпуске для канала Office Monthly Channel (Targeted)

Эта статья содержит заметки о выпуске для сборок приложений Word, Excel, PowerPoint, Outlook, Access и Project для Windows, предоставляемых в рамках программы предварительной оценки. Каждую неделю мы будем сообщать об интересных новых возможностях, важных исправлениях и существенных проблемах, о которых вам следует знать. Обратите внимание на то, что развертывание возможностей (а иногда даже исправлений) для участников программы предварительной оценки зачастую занимает определенное время. Благодаря этому мы обеспечиваем стабильную работу возможностей до того, как они становятся доступны более широкой аудитории. Если вы не видите чего-либо из описанного ниже, не беспокойтесь, вы получите это позже.  

> [!NOTE]
> - Заметки о выпуске публикуются еженедельно и могут представлять собой компиляцию для нескольких сборок
> - Дата публикации заметок о выпуске может не совпадать с фактической датой выпуска сборки
> - Microsoft Teams для существующих установок Office 365 профессиональный плюс — с конца июня Microsoft Teams будет добавляться в существующие установки Office 365 профессиональный плюс (и Office 365 бизнес) при обновлении этих установок. Дата добавления приложения Teams зависит от используемого канала обновления. Дополнительные сведения см. в статье [Развертывание Microsoft Teams с Office 365 профессиональный плюс](https://docs.microsoft.com/ru-RU/deployoffice/teams-install).

### <a name="resolved-issues"></a>Решенные проблемы
### <a name="access"></a>Access

- Исправлена проблема, из-за которой пользователи могли сталкиваться с ошибкой "Несогласованное состояние" при использовании общей базы данных.

### <a name="excel"></a>Excel

- Решена проблема производительности асинхронных пользовательских функций, из-за которой они выполнялись синхронно.
- Решена проблема, из-за которой книги, созданные в более ранних версиях Office, могли приводить к зависанию Excel при открытии в текущих версиях Office.
- Исправлена проблема, которая могла мешать пользователям вставлять гиперссылки в некоторых защищенных листах.
- Исправлена проблема, из-за которой имя шрифта на ленте могло отличаться от используемого шрифта.
- Решена проблема, из-за которой не удавалось правильно применять формат гиперссылки к некоторому содержимому.
- Решена проблема, из-за которой формулы со структурированными абсолютными ссылками могли изменяться неправильно.
- Решена проблема, из-за которой содержимое, скопированное из Excel, могло отображаться неправильно при вставке в другие приложения Office.
- Исправлена проблема, которая могла приводить к неправильному отображению точечных графиков при изменении набора рядов.
- Исправлена проблема, из-за которой Excel иногда зависал при запуске.
- Решена проблема со вставкой файлов в качестве объектов из OneDrive.
- Исправлена проблема, которая могла блокировать обновление сводных таблиц во время совместного редактирования.
- Исправлена проблема, из-за которой при предварительном просмотре неправильно отображалась область печати.
- Исправлены цвета, используемые при предварительном просмотре, когда вставляются диаграммы с использованием шаблонов диаграмм.

### <a name="onenote"></a>OneNote

- Исправлена проблема, из-за которой выполняется переход на исходную страницу по ссылке на скопированную страницу.

### <a name="outlook"></a>Outlook

- Исправлена проблема, которая приводила к неправильному потреблению ресурсов в Outlook при отключении защищенного режима для ограниченных сайтов в Internet Explorer.
- Исправлена проблема, которая могла мешать синхронизации почтовых ящиков клиентов с почтовыми ящиками в Outlook при переходе на современную проверку подлинности в Office 365.
- Исправлена проблема, из-за которой некоторые символы в метках подписей не отображались в выпадающем меню
- Значительно улучшена производительность при выборе помещения, если доступно много помещений.
- Исправлена проблема, которая могла мешать пользователям добавлять вложения в календари.
- Исправлена проблема, которая могла приводить к появлению сообщений об ошибке при ответе на сообщение с цифровой подписью.
- Исправлена проблема, из-за которой результаты расширенного поиска рейсов могли не отображаться в результатах поиска.
- Исправлена проблема, из-за которой пользовательский интерфейс мог зависать в компактном представлении.
- Исправлена проблема, из-за которой некоторые пользователи неправильно отображались в расписании группы как находящиеся вне сети.
- Исправлена проблема, из-за которой иногда при вставке текста из источника ANSI появлялись символы Юникода.
- Исправлена проблема, которая могла приводить к дублированию почтовых папок.
- Исправлена проблема, из-за которой могло неправильно возникать сообщение об ошибке при попытке отправить письмо, зашифрованное с помощью S/MIME.
- Исправлена проблема, из-за которой иногда мог возникать сбой при получении пользователем сообщения "Пропущенная беседа" из Skype.
- Исправлена проблема, которая могла приводить к утечке памяти.
- Исправлена проблема, из-за которой могло появляться сообщение об ошибках разрешений при взаимодействии с общими папками календаря.
- Исправлена проблема, которая могла приводить к изменению имени отправителя при сохранении в виде черновика.

### <a name="powerpoint"></a>PowerPoint

- Исправлена проблема, из-за которой у пользователя могла возникать ошибка при печати в PDF.
- Обнаружена проблема, из-за которой устройства ARC могли терять подключение при работе в AirSpace WinComp.
- Исправлена проблема, не позволявшая создавать гиперссылку при вставке текста с гиперссылкой.
- Исправлена проблема, которая могла мешать правильной работе статистических данных.
- Исправлена проблема, из-за которой могли повреждаться объекты TextRange после вставки текста в колонтитулы или заполнители номеров страниц слайдов в образце слайдов и макете слайда.

### <a name="project"></a>Project

- Исправлена проблема, которая могла приводить к сбою при замене корпоративного ресурса локальным ресурсом.
- Исправлена проблема с сохранением локальных ресурсов, тарифов и других свойств подпроектов при их сохранении в Project Web App с помощью главного проекта.
- Исправлена проблема, из-за которой изменение значения трудозатрат в суммарной задаче могло приводить к сбою при включенной функции защищенных значений трудозатрат.
- Исправлена проблема, из-за которой некоторые сведения о задаче сохраняются с ошибками при копировании или вставке локальных значений настраиваемого поля из главного проекта во вставленный проект.
- Исправлена проблема, которая могла блокировать возможность синхронизации событий с корпоративными календарями.

### <a name="word"></a>Word

- Исправлена проблема, которая могла приводить к сбою сочетания клавиш CTRL+V.
- Исправлена проблема, которая могла мешать правильной работе синхронной прокрутки в режиме черновика.
- Исправлена проблема, из-за которой могли возникать проблемы с масштабированием при печати на принтерах Deskjet.
- Исправлена проблема, из-за которой всплывающие подсказки могли неправильно отображаться после сохранения документа в первый раз.
- Исправлена проблема с потерей форматирования таблиц.
- Решена проблема со вставкой файлов в качестве объектов из OneDrive.
- Исправлена проблема, из-за которой не фиксировался цвет шрифта.

### <a name="office-suite"></a>Набор приложений Office

- Исправлена проблема, из-за которой при закрытии ожидающего отправки файла у пользователя могло отображаться неверное сообщение об ошибке.
- Исправлена проблема, которая могла приводить к повторному предупреждению об удалении локальных версий файла.
- Исправлена проблема, из-за которой мог предлагаться журнал версий, когда эта функция отключена.
- Теперь имя для входа предварительно заполняется для основных запросов проверки подлинности в Outlook.
- Исправлена проблема, из-за которой к полужирному тексту среднего размера мог быть применен неправильный стиль.
